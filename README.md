# 🔧 Sensor Data Analytics (IoT)

## 📌 Overview

Analyzed time-series sensor data to detect anomalies and monitor system health.

## 🛠️ Tools

* Python
* Pandas
* NumPy
* Matplotlib

## 🔍 Key Features

* Time-series analysis
* Z-score based anomaly detection
* Multi-sensor correlation analysis

## 🚨 Outcome

* Detected abnormal sensor behavior
* Built a basic predictive maintenance system
